UberCart 3 Payment Gateway for Heartland SecureSubmit

Installation and Setup
======================

a) Add the SecureSubmit folder to the drupal modules folder.

b) Navigate to the store's Administration page, Configuration section, and enable the gateway under the Payment Gateways. (admin/store/settings/payment/edit/gateways)

c) Configure your public and secret API keys.

d) Download and install the latest version of the SecureSubmit PHP SDK from https://github.com/SecureSubmit/heartland-php and place it in sites/all/libraries/securesubmit
